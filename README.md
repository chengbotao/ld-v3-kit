# LD V3 Kit

Vue3 组件库，基于 Element Plus 构建。

## 安装

```bash
npm install ld-v3-kit
```

## 快速开始

```ts
import { createApp } from 'vue'
import App from './App.vue'
import ElementPlus from 'element-plus'
import LdV3Kit from 'ld-v3-kit'
import 'ld-v3-kit/styles/index.css'

const app = createApp(App)
app.use(ElementPlus)
app.use(LdV3Kit)
app.mount('#app')
```
